REST API Words

Esta API te permite obtener palabras aleatorias.

Endpoints v1

  • GET /api/v1/words
    Devuelve una palabra aleatoria.
  • GET /api/v1/words?length=3
    Devuelve una palabra aleatoria de 3 letras (entre 3 y 13).
  • GET /api/v1/words/all
    Devuelve todas las palabras disponibles en la base de datos (incluyendo las nuevas guardadas).

Ejemplo de respuesta v1

{
  "word": "mar"
}

Endpoints v2

Nuevas funcionalidades con soporte para múltiples idiomas y conexión a fuentes externas.

  • GET /api/v2/languages
    Devuelve la lista de códigos de idioma soportados (ej: 'es', 'en', 'it').
  • GET /api/v2/words?length=5&lang=es
    Devuelve una palabra aleatoria filtrada por longitud e idioma.
    Parámetros: length (número) y lang (código de idioma).

Ejemplos de respuesta v2

Idiomas: 
{
  "languages": [
    "zh",
    "pt-br",
    "es",
    "de",
    "it",
    "fr"
  ]
}
Palabra (Proxy): 
{
  "word": "gato"
}

Integraciones Externas (Prácticas)

Endpoints conectados a APIs públicas populares.

  • GET /api/v2/jokes/chucknorris
    Devuelve una broma de Chuck Norris (Categoría: Dev).
  • GET /api/v2/pokemon/pikachu
    Devuelve información básica de Pikachu (PokeAPI).
  • GET /api/v2/trivia/history
    Devuelve 3 preguntas de trivial sobre Historia.
  • GET /api/v2/weather/barcelona
    Devuelve el clima actual de Madrid (OpenWeatherMap / Open-Meteo).

Ejemplo Clima (JSON)

{
  "city": "Madrid",
  "temperature": 18.5,
  "source": "OpenWeatherMap (Autenticado)"
}
